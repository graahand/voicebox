## packages required to be installed in docker (jetson)

apt-get install ffmpeg espeak-ng libsndfile1 portaudio19-dev mecab libmecab-dev mecab-ipadic-utf8 libsox-fmt-all sox alsa-utils


## docker image commit command

sudo docker commit 'container_id' docker-image name

For instance, sudo docker commit 34ljldfjlak sdcsatyam:v5
