# Futuruma 2026 - Knowledge Base

This document contains structured information about Futuruma 2026, the biggest tech fest in Nepal.

---

## Event Overview

### What is Futuruma?

Futuruma is the biggest tech fest in Nepal organized by ING Skill Academy. It is held in different cities across Nepal to educate, inspire, and make students aware of career opportunities in the IT and business sectors.

### Event Organizer

The event is organized by ING Skill Academy in collaboration with Skill Museum. 

### Event Purpose and Aim

The main aim of Futuruma is to:
- Educate students about emerging technologies
- Inspire the next generation of tech professionals
- Create awareness about career opportunities in IT and business sectors
- Showcase innovative projects and technologies

## Event History

### First Futuruma Event (2024)

The first Futuruma event was held in 2024 in 10 different cities of Nepal. During this event:
- Projects were showcased to students across Nepal
- Around 50,000 students participated
- The event spanned 20 days
- Students were educated about IT sector and business sector careers

### Participation Statistics

In 2024, approximately 50,000 students participated in the Futuruma event across all cities.

## Robotics Projects

All robotics projects are developed by Skill Museum.

### Self Balancing Robot

A robot that can balance itself on two wheels using sensors and control algorithms. Developed by Skill Museum.

### Line Following Robot

A robot that follows a predefined path marked on the ground using sensors. Developed by Skill Museum.

### Obstacle Avoidance Robot

A robot equipped with sensors to detect and navigate around obstacles. Developed by Skill Museum.

### Robotic Arm

A programmable mechanical arm for picking and placing objects. Developed by Skill Museum.

### Laser Tag

Laser Tag is a real-world First Person Shooter game played in teams of two players. Each player has 3 lives. The game uses laser technology for shooting. Developed by Skill Museum.

---

## Artificial Intelligence Projects

All AI projects are developed by Skill Museum.

### AI Maths Assistant

An AI-powered assistant for answering mathematics-related queries. Designed specifically for students from Nepal. Developed by Skill Museum.

### DermaScan / Skin Health Analysis

A computer vision project that can:
- Predict skin conditions
- Estimate skin age
- Analyze face shape
Developed by Skill Museum.

### Kick and Crush

An interactive game that replicates Fruit Ninja using Mediapipe keypoint detection API. Players use body movements to slash virtual fruits. Developed by Skill Museum.

---

## Cybersecurity Projects

All cybersecurity projects are developed by Skill Museum.

### Cybercentric Island

An open-world 3D game based on the structure of Islington College. The game includes cybersecurity challenges that players must solve. Developed by Skill Museum.

### Bhavisyawani / AstroSpy

A cybersecurity awareness project demonstrating how hackers and suspicious websites can access user cameras through social engineering. The project uses a fortune-telling (bhavisyawani) theme to demonstrate these techniques. Developed by Skill Museum.
