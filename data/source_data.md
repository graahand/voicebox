# Futuruma 2026

## Organizer of the event is ING Skill Academy


## what is Futuruma?

Futuruma is the biggest tech fest in Nepal organized by ING Skill Academy in different cities of Nepal. 

## Aim of Futuruma
The main aim of this event is to educate, inspire and aware the students from all over the country. 

## Types of projects shown in futuruma 

Technologies like Cybersecurity, artificial intelligence, software development, game development, robotics, mobile application development, networking are covered in this tech fest. 

## Venue List for Futuruma

kathmandu, itahari, biratnagar, pokhara, chitwan, butwal, birgunj



## Robotics Projects that are being showcased at Futuruma 

### Self Balancing Robot developed by Skill Museum

### Line Following Robot developed by Skill Museum

### Obstacle Avoidance Robot developed by Skill Museum

### Robotics Arm developed by Skill Museum

### Laser Tag 
Laser tag is real world First Person Shooter game played in a team of two with 3 lives per player. It was developed by Skill Museum 


## AI Projects

### AI Maths Assistant

Project for any maths related queries for students from Nepal. It was developed by Skill Museum 


### DermaScan/Skin Health Analysis 

this project uses computer vision to predict skin condition, skin's age and face'shape. It was developed by Skill Museum 


### Kick and Crush 

this project is replication of Fruits Ninja Game with the help of Mediapipe keypoints detections API for interactive experience. It was developed by Skill Museum 


## CyberSecurity Projects

### Cybercentric Island 

this project is an open-world game based on the 3D structures of Islington College and Cybersecurity challenges. It was developed by Skill Museum 


### Bhavisyawani/AstroSpy 

this project is based on how the hackers or suspicious websites access our cameras by emotionally engineering us (in this case bhavisyawani (future prediction). It was developed by Skill Museum 


## History of Futuruma

The fist futuruma event was held on 2024 in 10 different cities of Nepal where we showcased the projects, inspired and educated about the prospective career on IT sector and Business Sector in Nepal to around 50,000 students in 20 days. 


### Number of Participations in 2024
